Installation Methods

Multiple ways to overcome the 32-bit EFI challenge

‚≠ê macOS Installer

Easiest

Best for: MacBooks still running Snow Leopard (10.6) through Mountain Lion (10.8)

Time: 15-20 minutes

  1. Download bootstrap tools
  2. Transfer to MacBook via USB
  3. Run installer from macOS Terminal
  4. Reboot - now accepts LiveUSBs!
Pros: No hardware disassembly, works from within macOS
Requires: Working macOS installation

üîß SSD Swap Method

Most Reliable

Best for: Any situation, especially if macOS is broken

Time: 30-45 minutes

  1. Remove SSD from MacBook
  2. Install in modern computer
  3. Boot Thyme OS LiveUSB with forced 32-bit EFI
  4. Install normally, move SSD back to MacBook
Pros: Always works, no dependency on macOS
Requires: SSD removal, second computer

üåê Network Installation

Advanced

Best for: Multiple MacBooks, advanced users

Time: 60 min setup, 30 min per MacBook

  1. Set up network boot server
  2. Connect MacBook via Ethernet
  3. Boot holding 'N' key
  4. Install over network
Pros: No hardware disassembly, scalable
Requires: Ethernet, complex setup
